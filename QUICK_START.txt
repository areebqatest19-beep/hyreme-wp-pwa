🆘 IF SOMETHING BREAKS

1. Check WordPress debug log:
   tail -f wp-content/debug.log
   
2. Enable WordPress debugging in wp-config.php:
   define('WP_DEBUG', true);
   define('WP_DEBUG_LOG', true);
   
3. Check browser console (F12):
   - Console tab: Any red errors?
   - Network tab: Failed requests?
   - XHR filter: Check AJAX responses

4. Most common issues:
   • Upload crashes? → chmod 755 wp-content/uploads/
   • "Failed to fetch"? → Check nonce name matches
   • Data not saving? → Query user meta for conv_ keys
   • Slow? → Check debug log, clear cache
